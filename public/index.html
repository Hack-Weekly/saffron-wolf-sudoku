Saffron Wolf Sudoku Puzzler

Click on each of the squares to fill them with desired values. Then click on the "SOLVE" button to get the solution to the sudoku puzzle. The app will check for valid inputs as the user enters values into the squares & the square will flash red if value entered isn't allowed as per sudoku rules. If the values entered do not result in a valid or solvable sudoku the app will generate an error stating "no solution found!"
